This is a collection of some of the projects I have been part of during my studies.
Not all of them have code available.

## DAT250 - Poll app

This is a group project for the course DAT250 - Advanced Software Technologies. The goal of the
project was to create a web application with a backend and a frontend. The backend was implemented
using C# and ASP.NET Core and the frontend was implemented using React.

## DAT251 - Rideshare app

This is a group project for the course DAT251 - Modern Software Development Methods. In this course
we created a phone application for people to share rides with each other. The course itself was about
agile software development where we used methodologies from Scrum and XP to develop the application.
The application was developed using React Native for the phone client and Spring Boot for the backend.
